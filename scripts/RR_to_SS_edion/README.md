# RR_to_SS_edion

## 概要

楽楽販売APIからフォーム経由などで入力されたエディオン案件の問い合わせデータをCSV形式で取得し、Googleスプレッドシートの「元データ管理シート（マスタDB）」へ洗い替え（C2セル以降を全クリア後に再書き込み）で反映するスクリプトです。

---

## 処理の流れ

```
楽楽販売（CSVエクスポートAPI）
   ↓ 問い合わせ管理DBから最大5,000件取得（dbSchemaId: 101181）
列マッピングと整形
   ↓ 必要な18項目に絞り込み、住所列は複数フィールドを結合
   ↓ 全セルの前後空白・先頭シングルクォートを除去
Googleスプレッドシート更新
   ↓ C2:Z の既存データをクリア
   ↓ C2セルから全データを一括書き込み（USER_ENTERED形式）
完了ログ出力
```

---

## 詳細な処理内容

### 1. 楽楽販売からのデータ取得

- DB: 問い合わせ管理（`dbSchemaId: 101181`, `searchId: 107497`, `listId: 101473`）
- 最大5,000件を取得
- レスポンスは CP932 でデコード（失敗時は UTF-8 で再試行）

### 2. 列マッピングと整形

スプレッドシートのC列以降（18項目）に以下の順序でデータを配置します：

| スプレッドシート列 | CSV列名 | 備考 |
|---|---|---|
| C列: ステータス | ※固定値 | 常に「新規」を書き込み |
| D列: レコードID | 記録ID | |
| E列: エディオン連携用手配番号 | エディオン連携用手配番号 | |
| F列: 問い合わせ日 | 新規問い合わせ日 | |
| G列: 顧客名 | 名前 | |
| H列: 顧客名カナ | フリカナ | |
| I列: 顧客電話番号 | 電話番号_1 | |
| J列: 顧客メールアドレス | メールアドレス_1 | |
| K列: 顧客住所 | 郵便番号+都道府県+市区+町村+マンション | 5列を結合 |
| L列: （空白） | ― | |
| M列: ガス種 | ガスの種類 | |
| N列: 先方店舗名 | エディオン店舗名 | |
| O列: 先方担当者名 | エディオン担当者名 | |
| P列: 先方メールアドレス | エディオンメールアドレス | |
| Q列: 工事第１希望日 | 第１希望日（エディオン） | |
| R列: 工事第２希望日 | 第２希望日（エディオン） | |
| S列: 工事第３希望日 | 第３希望日（エディオン） | |
| T列: スグフォームURL | スグフォームURL（エディオン） | |

**住所結合ロジック**: 以下の5列を順に結合（空のフィールドはスキップ）
1. `エディオン案件用：郵便番号`
2. `都道府県名`
3. `市区名`
4. `町村名・番地`
5. `マンション・ビル名`

**データクリーニング**: 全セルに対して `strip()` + `lstrip("'")` を実施

### 3. スプレッドシート更新

- A列・B列および1行目（ヘッダー行）は**変更しない**
- `worksheet.batch_clear(['C2:Z'])` で既存データを全クリア
- `worksheet.update(range_name='C2', value_input_option='USER_ENTERED')` で一括書き込み
- `USER_ENTERED` モードにより、日付文字列・数値文字列が自動変換される

---

## 手動管理列（スクリプトで上書きしない）

以下の列はスプレッドシート上で手動管理されるため、スクリプトは書き込みを行いません。
スプレッドシートのA列・B列がそのまま保持されます。

- 新規フラグ / 先方確認フラグ / ステータス / 見積作成日 / 先方確認日 / 備考 / 見積金額 / 見積書URL

## 対象DB情報

| 項目 | 値 |
|---|---|
| スキーマID | `101181` |
| 検索ID | `107497` |
| 一覧ID | `101473` |
| 対象スプレッドシート | `元データ管理シート（マスタDB）` |

---

## 環境変数（必須）

| 変数名 | 説明 |
|---|---|
| `RAKURAKU_TOKEN` | 楽楽販売 APIトークン |
| `GOOGLE_SERVICE_ACCOUNT_JSON` | GCPサービスアカウントのJSON文字列 |

> ローカル実行時は同フォルダの `.env` ファイルに記述してください。

---

## 実行方法

```bash
python scripts/RR_to_SS_edion/RR_to_SS_edion.py
```

ログは同フォルダの `execution_log.txt` に追記されます。
