# tenki_to_SS_tempreture

## 概要

気象庁のWebサイトから前日の**全都道府県（47都道府県）の最低気温**を自動スクレイピングし、Googleスプレッドシートの日付列へ書き込むスクリプトです。

---

## 処理の流れ

```
実行日の前日を計算
   ↓ 書き込み先シート名・列位置を決定（例: 「1月_new」シートのN日目の列）
気象庁Webサイトをスクレイピング
   ↓ 47都道府県の主要観測地点から月間日別データを取得（city×1リクエスト）
最低気温を抽出
   ↓ 各都道府県の前日の最低気温を取得（取得できない場合は「-」）
Googleスプレッドシートへ書き込み
   ↓ 縦1列（47行分）を一括書き込み → 右寄せ書式を適用
```

---

## 詳細な処理内容

### スクレイピング元

気象庁 過去の気象データ検索（月間日別データページ）  
`https://www.data.jma.go.jp/obd/stats/etrn/view/daily_s1.php`

各都道府県の代表都市コード（`prec_no`・`block_no`）を設定済みです。

### 書き込み位置

- **基準セル**: `BASE_CELL = "B62"` （北海道の1日分が入る起点）
- 対象日が N 日の場合、基準列から (N-1) 列ずらした列へ書き込み
- シート名は `{月}月_new` 形式（例: `1月_new`）

### 取得データの構造

気象庁の月間データページから `pd.read_html()` で取得し、**0列目（日付）・8列目（最低気温）** のみを使用します。

---

## 設定値（スクリプト内）

| 変数 | 内容 |
|---|---|
| `SPREADSHEET_KEY` | 書き込み先スプレッドシートID |
| `BASE_CELL` | 北海道1日分の基準セル（デフォルト: `B62`） |
| `ALL_CITIES` | 47都道府県の気象庁観測地点コードリスト |

---

## 環境変数（必須）

| 変数名 | 説明 |
|---|---|
| `GOOGLE_SERVICE_ACCOUNT_JSON` | GCPサービスアカウントのJSON文字列 |

---

## 実行方法

```bash
python tenki_to_SS_tempreture.py
```

スクレイピングのサーバー負荷軽減のため、都道府県ごとに **0.1秒のスリープ** を入れています。
