# RR_to_SS_zaiko

## 概要

楽楽販売の在庫管理DBから全在庫データをCSVで取得し、Googleスプレッドシートの「楽楽販売在庫表」シートへ**全件上書き（洗い替え）**するスクリプトです。

---

## 処理の流れ

```
楽楽販売（CSVエクスポートAPI）
   ↓ 在庫管理DBの全データを最大5,000件取得（dbSchemaId: 101296）
データ整形
   ↓ 全セルの前後空白（strip）・先頭シングルクォート（lstrip("'")）を除去
Googleスプレッドシート更新
   ↓ 対象シートを clear() で完全クリア
   ↓ A1から全データを一括書き込み（USER_ENTERED形式）
完了ログ出力
```

---

## 詳細な処理内容

### 1. 楽楽販売からのデータ取得

- 対象DB: 在庫管理（`dbSchemaId: 101296`）
- 絞り込み設定: `searchId: 104283`、一覧設定: `listId: 101100`
- 最大5,000件を取得
- レスポンスは UTF-8 でデコード（エラーは ignore で無視）

### 2. データ整形

全行・全セルに対して以下の処理を実施:

```python
clean_cell = cell.strip().lstrip("'")
```

- `strip()`: 前後の空白・改行を除去
- `lstrip("'")`: 先頭のシングルクォートを除去（楽楽販売がCSVに付与する場合がある）

これにより、日付文字列や数値文字列が「生のデータ」になります。

### 3. スプレッドシート書き込み

```python
worksheet.clear()  # シートを完全クリア
worksheet.update(range_name='A1', values=data_list, value_input_option='USER_ENTERED')
```

- `clear()` で既存データをすべて削除（ヘッダーも含む）
- A1セルから全データ（ヘッダー行込み）を一括書き込み
- `USER_ENTERED` モードにより以下の自動変換が行われる:
  - 日付文字列（例: `2024/01/15`）→ スプレッドシートの日付データ
  - 数値文字列（例: `123`）→ 数値データ（`001` は `1` になることに注意）

---

## 設定値（スクリプト内）

| 変数 | 値 | 内容 |
|---|---|---|
| `RAKURAKU_SCHEMA_ID` | `101296` | 楽楽販売 在庫DB Schema ID |
| `RAKURAKU_SEARCH_ID` | `104283` | 絞り込み設定ID |
| `RAKURAKU_LIST_ID` | `101100` | 一覧画面設定ID |
| `SPREADSHEET_KEY` | ※スクリプト内で設定 | 書き込み先スプレッドシートID |
| `SHEET_NAME` | `楽楽販売在庫表` | 書き込み先シート名 |

---

## 環境変数（必須）

| 変数名 | 説明 |
|---|---|
| `RAKURAKU_TOKEN` | 楽楽販売 APIトークン |
| `GOOGLE_SERVICE_ACCOUNT_JSON` | GCPサービスアカウントのJSON文字列 |

> ローカル実行時は同フォルダの `.env` ファイルに記述してください。

---

## 実行方法

```bash
python RR_to_SS_zaiko.py
```

ログは同フォルダの `execution_log.txt` に追記されます。

---

## 注意事項

- 本スクリプトは**洗い替え（全件削除→全件挿入）**方式を採用しています。既存データはすべて上書きされます。
- `USER_ENTERED` モードでは `001` のような先頭ゼロの数値が `1` に変換されます。先頭ゼロを保持したい場合はスプレッドシートの列を「書式なしテキスト」に設定してください。
